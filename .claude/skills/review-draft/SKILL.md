---
Create an agent team to review this manuscript draft $ARGUMENTS.

Phase 1 (parallel):
- Lead: extract the core argument and contribution claim
- Skeptic: search for related and competing work
- Connector: search for complementary work in adjacent fields

Phase 2 (depends on Phase 1):
- Methodologist: evaluate soundness and generalizability
- Visionary: assess impact and venue fit
- Newcomer: flag clarity issues, jargon, and accessibility

Phase 3 (depends on Phase 2):
- All: debate whether this is publishable and where. Produce structured feedback organized by dimension.
